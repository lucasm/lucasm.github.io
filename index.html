About me

Hello. I’m Lucas Menezes, a full-stack web developer based in Belém of Pará, Brazil. I’ve had the privilege of work with small and medium organizations in the last years, building mobile web apps, PWAs and responsive websites with CMS and Ecommerce capabilities — helping them establish a refined digital presence.

Creating fast, accessible and useful digital products across the multiple devices and browsers is something I really enjoy doing.

The journey of solve problems with programming brought me here: a professional specialized in front-end (HTML, CSS / Sass, SVG, Bootstrap, JavaScript / jQuery, React) who is capable to pull the back-end up (PHP / Yii, Node.js / Express, data modeling, SQL, Git, API integration, cloud services).

Recently, I’m learning data science, and playing with VR/AR immersive web experiences. Feel free to check my work and open source projects below.
